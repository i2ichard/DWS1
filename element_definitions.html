Element Definitions

<h1>
This is the Main Level Header.
<h2>
This is a Level 2 header.
<h3>
This is a Level 3 header.
<h4>
This is a Level 4 header.
<strong>
This generally makes the content BOLD.
<em>
This generally makes the content Italic to show emphesis.
<i>
Any word or phrase within these tags will be in italic.
<b>
Any word or phrase within these tags will be in BOLD.
<blockquote>
This places a quote in its own blocked off section.
<s>
This puts a strike through.
<sup> and <sub>
Any thing within these tags will either put it in a superscript or a subscript.
<p></p>
These are the opening and closing tags for a paragraph.
<ul></ul>
These are the opening and closing tags for an unordered list.
<ol></ol>
These are used in an ordered list.
<li></li>
These are the opening and closing tags for items in a list.
<dl>
This is a definition list.
<dt>
This is the term being defined.
<dd>
This is the definition of the term.
